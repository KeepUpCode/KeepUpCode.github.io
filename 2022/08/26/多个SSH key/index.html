多个 SSH key 的配置

KeepUP
2022-08-26
git, ssh

生成 ssk key

使用命令生成 ssh key，注意需要指定名字。

比如你想生成 Github 用的，可以给他起个名字 github_id_rsa

然后再生成一个 Gitlab 用的 key,名字为 gitlab_id_rsa

如果你有两个 Github 账号，你也可以再生成一个 github1_id_rsa

添加 config 配置

在 ~/.ssh 目录下，新建一个 config 文件，然后配置以下内容

  • Host 自定义别名，后边会用到
  • HostName 服务器域名
  • 用户名，有的是邮箱,一般默认情况 git 就可以
  • IdentityFile 对应的私钥,也就是不带.pub 后缀的

使用

你以前用 ssh 克隆项目时是：git clone git@github.com:username/repository.git

但是现在你需要使用 config 里边定义的 Host，一般情况下 Host 和服务器域名的名字相同，但是比如我们有两个 github 账号，我第二个账号的别名是 github.A.com ，那我们克隆的时候就需要改成
